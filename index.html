What I can do.

Analyse Data.

I use python and jupyter notebook to analyze raw data, mining for insights, and to transform metrics, facts, and figures into initiatives for improvement.

Train AI and ML models.

I mainly use python, and rarely use javascript to build and train my AI and ML models, they are at most 3-4 lines of code, it's all about choosing the right combo.

Develop Web Applications.

To build a business online, to build an online presence, or to deliver services online, websites are the way to go. I use the MERN stack to build neat websites.

Develop Mobile Applications.

Mobile applications are 1up to websites. They are more customizable, more flexible, and supports complex functionalities. I use React Native to build awesome apps.

Automate Mundane tasks.

Sometimes the process of doing something can be simple but daunting, they can be repetitive, like collecting data from the internet, they are better automated.

Hunt Bugs and Build Tests.

Mostly I follow TTD to avoid bugs, occasionally when the project is small, or time is of the essence I just wing it and then go on a bug hunt. I use jest and selenium.

Code Games and Simulations.

Writing logic for a game is more fun than playing the game itself, it's like piecing a jigsaw puzzle. I use P5js for simple games and Unity for more complex ones.

Animate 3D Models.

I use Maya to breathe life into a well-structured clump of polygons. Animation takes more time and it needs attention to detail, but its fun all the way.

Build Robots.

Robot building can be so much fun, designing circuits, assembling parts, and programming its functions. I use HDL for designing circuitry, and Arduino for functionality.
